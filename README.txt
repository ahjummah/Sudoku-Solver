This is a SudokuSolver program that solves a sudoku puzzle using Genetic Algorithms.

Before running the program in Netbeans, the user can change which puzzle would be
solved by the program. The file initializer can be found in the main class, it looks
like this:  SudokuSolverReal s = new SudokuSolverReal("sudoku4_01.txt");

Here are the alternative sudoku puzzle filenames:
- sudoku4_01.txt
- sudoku4_03.txt
- sudoku6_01.txt
- sudoku6_02.txt
- sudoku8_01.txt
- sudoku8_02.in
- sudoku9_01.in
- sudoku9_02.in
